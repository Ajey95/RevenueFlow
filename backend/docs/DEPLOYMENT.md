# RevenueFlow Backend

The world's first autonomous revenue operations agent powered by Fetch.ai, Groq + Llama, running on Vultr infrastructure.

## Quick Start

```bash
# Install dependencies
npm install

# Set up environment
cp .env.example .env
# Edit .env with your API keys

# Start development server
npm run dev

# Test services
npm run test:groq
npm run setup:agents
